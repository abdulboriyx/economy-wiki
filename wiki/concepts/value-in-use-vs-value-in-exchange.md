# Value in Use vs Value in Exchange

## Definition
- **Value in Use:** The utility or benefit a person gets from using a specific item.
- **Value in Exchange:** The power of an item to purchase other goods, typically expressed as a market price.

## Distinguishing from related concepts
This distinction is often illustrated by the **"Diamond-Water Paradox"**: water is essential for life (high value in use) but is often very cheap or free (low value in exchange), whereas diamonds have little practical use (low value in use) but command a very high price (high value in exchange). It differs from modern utility theory, which resolves this by looking at "marginal utility" (the value of the *next* unit consumed).

## Real-world examples
- **Air:** Extremely high value in use but zero value in exchange (under normal conditions).
- **Crypto-currency:** High value in exchange (at times) with debated value in use.
